# Week 1 - Exercise 6: Stored Procedures (PL/SQL)

## Objective

To implement stored procedures in PL/SQL for common banking operations.

## Files

* StoredProcedures.sql

## Scenarios Implemented

### Scenario 1

Created the `ProcessMonthlyInterest` procedure to apply 1% monthly interest to all savings accounts.

### Scenario 2

Created the `UpdateEmployeeBonus` procedure to update employee salaries based on department and bonus percentage.

### Scenario 3

Created the `TransferFunds` procedure to transfer funds between two accounts after checking sufficient balance.

## Concepts Used

* Stored Procedures
* Procedure Parameters
* Variables
* IF Condition
* UPDATE Statement
* COMMIT
* DBMS_OUTPUT.PUT_LINE

## Result

Successfully created, executed, and verified all three stored procedures using Oracle Live SQL.
