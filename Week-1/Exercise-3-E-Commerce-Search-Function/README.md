# Week 1 - Exercise 3: E-Commerce Platform Search Function

## Objective

To implement Linear Search and Binary Search algorithms for searching products in an e-commerce platform and compare their performance.

## Files

* Product.java
* SearchAlgorithms.java
* SearchTest.java

## Algorithms Used

### Linear Search

* Searches each product one by one.
* Works on unsorted arrays.
* Time Complexity: **O(n)**

### Binary Search

* Searches a sorted array by repeatedly dividing the search space into two halves.
* Time Complexity: **O(log n)**

## Output

```text
===== Linear Search =====
Product ID: 104, Name: Watch, Category: Accessories

===== Binary Search =====
Product ID: 104, Name: Watch, Category: Accessories
```

## Conclusion

Both search algorithms were implemented successfully. Linear Search is suitable for small or unsorted datasets, while Binary Search is faster for large sorted datasets because it has lower time complexity.
