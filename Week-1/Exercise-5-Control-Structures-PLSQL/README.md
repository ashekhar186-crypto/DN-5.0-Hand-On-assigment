# Week 1 - Exercise 5: Control Structures (PL/SQL)

## Objective

To implement PL/SQL control structures for solving banking-related scenarios using loops, conditional logic, and SQL statements.

## Files

* ControlStructures.sql

## Scenarios Implemented

### Scenario 1

Applied a 1% discount to loan interest rates for customers whose age is greater than 60 years.

### Scenario 2

Updated the `IsVIP` flag to `TRUE` for customers whose account balance is greater than 10000.

### Scenario 3

Displayed reminder messages for customers whose loan due dates fall within the next 30 days.

## Concepts Used

* PL/SQL Blocks
* FOR Loop
* UPDATE Statement
* COMMIT
* DBMS_OUTPUT.PUT_LINE
* Date Functions

## Result

Successfully implemented and verified all three PL/SQL scenarios using Oracle Live SQL.
